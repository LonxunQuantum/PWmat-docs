# Q-Studio

Q-Studio是一款原子尺度的材料建模应用，主要用于辅助用户进行计算前的结构准备工作。Q-Studio支持小分子、晶体、纳米团簇、无序结构、异质结构、过渡态结构、量子输运器件等模型的建模，并可导出模型到Q-Flow结构库或本地。登录[Mcloud:龙讯超算云平台](https://mcloud.lonxun.com/)，点击 Q-Studio 图标打开应用，开始材料建模工作。
